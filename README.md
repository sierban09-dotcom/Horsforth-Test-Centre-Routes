# Driving Test Route Tracker 🚗

A simple web app to view and track driving test routes (Horsforth example included).

## Features
- Load predefined test routes
- GPS live tracking
- Click-to-create custom routes
- Save routes as JSON
- Map powered by OpenStreetMap

## How to run
Just open `index.html` in a browser.

## Tech stack
- HTML
- JavaScript
- Leaflet.js
- OpenStreetMap
